# Adoptme - แพลตฟอร์มสำหรับคนรักสัตว์

**Adoptme** เป็นแพลตฟอร์มที่รวบรวมผู้รักสัตว์ ผู้มีประสบการณ์เลี้ยงสัตว์ ศูนย์พักพิง และผู้เพาะพันธุ์ที่มีความรับผิดชอบ มารวมอยู่ในแพลตฟอร์มที่น่าเชื่อถือ โดยแพลตฟอร์มนี้จะเป็นพื้นที่สำหรับแลกเปลี่ยนความรู้เกี่ยวกับการดูแลสัตว์เลี้ยง และเป็นแหล่งที่เชื่อถือได้ในการรับอุปการะหรือส่งต่อสัตว์เลี้ยง เพื่อให้สัตว์ทุกตัวได้พบกับเจ้าของใหม่ที่พร้อมดูแล และได้รับคุณภาพชีวิตที่ดี

---

## ฟีเจอร์หลัก (Main Features)

- ผู้ใช้สามารถโพสต์ประกาศหาบ้านให้สุนัขหรือแมวจร
- สามารถค้นหาและกรองข้อมูลตามจังหวัดหรือประเภทสัตว์
- รองรับการอัปโหลดรูปภาพสัตว์
- มีระบบล็อกอินสมัครสมาชิกสำหรับผู้ใช้ทั่วไปและผู้ดูแลระบบ
- ผู้ดูแลสามารถจัดการโพสต์ที่ไม่เหมาะสมได้ผ่านหน้า Admin
- สามารถซื้อขายของใช้เกี่ยวกับสัตว์เลี้ยงได้
